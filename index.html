Welcome to the "Stay Organized" Workshop! Our team—Destiny, Monse, Ian, Adama, and Etson—is excited to guide you through this collaborative project where we built a dynamic website designed to help users manage their to-do lists efficiently. Our platform will feature a sleek home page, a comprehensive task viewing page, and a user-friendly interface for adding new tasks.

Utilizing a REST API for seamless data interaction, we have implemented the essentials of web development, from setting up a shared GitHub repository to following best practices in coding and design. With additional features to enhance functionality, our project fosters teamwork, communication, and practical coding skills in a real-world environment. Explore our tool and discover how staying organized can be simple and effective!
